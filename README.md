# work-hours-calculator
This is a small application wrote in python for working worksheet which can be apply for the project cost management or working hours tracking for your employee. The methodology for using this is to open the .exe file, then add the date using yyyymmdd and add check in, check out in HH:MM format. You can copy the previous line data to next line, or export/print.

The .exe file is in the compressed file.

The external libary used in this application is numpy, reportlab
